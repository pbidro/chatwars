# chatwars

## Dependencias
Este proyecto utiliza las librerías 
    
    firebase
    
    bootstrap vue
```
npm install firebase
npm install vue bootstrap bootstrap-vue
```

### Para compilar ejecutar el siguiente comando
```
npm run serve
```

### Para prerenderizar la aplicación
```
npm run build
```
